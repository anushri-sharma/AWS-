
# Module 8 - AI/ML & Data Analytics

- Convert speech into text.

4. [Amazon Translate](https://aws.amazon.com/translate/)

- Translate text into multiple languages.

5. [Amazon Kendra](https://aws.amazon.com/kendra/)

- Use natural language processing to intelligently query enterprise content.

6. [Amazon Rekognition](https://aws.amazon.com/rekognition/)

- Identify objects and activities in images and videos.

7. [Amazon Textract](https://aws.amazon.com/textract/)

- Detect and extract typed and handwritten text in documents.

8. [Amazon Lex](https://aws.amazon.com/lex/)

- Add voice and text conversational interfaces to applications.

9. [Amazon Personalize](https://aws.amazon.com/personalize/)

- Add personalized customer recommendations to applications.

10. [Amazon SageMaker AI](https://aws.amazon.com/sagemaker/ai/)

- Build, train, and deploy your own ML models without worrying about infrastructure.

11. [Amazon SageMaker JumpStart](https://aws.amazon.com/sagemaker/ai/jumpstart/)

- Deploy pre-trained ML solutions, like computer vision, NLP, and tabular data, with just a few clicks.

12. [Amazon Bedrock](https://aws.amazon.com/bedrock/)

- Fine-tune and seamlessly integrate large FMs from Amazon and leading AI startups into your AWS applications with a single API.

13. [Amazon Q Business](https://aws.amazon.com/q/business/)

- Answer questions and solve problems using the data and expertise found in your company's information repositories.

14. [Amazon Q Developer](https://aws.amazon.com/q/developer/)

- Accelerate development with code recommendations.

15. [Amazon Kinesis Data Streams](https://aws.amazon.com/kinesis/data-streams/)

- Ingest terabytes of streaming data in real time.

16. [Amazon Data Firehose](https://aws.amazon.com/firehose/)

- Ingest and deliver data within seconds to multiple consumers, such as data lakes and warehouses.

17. [Amazon S3](https://aws.amazon.com/s3/)

- Store virtually limitless amounts of all types of data using this popular data lake choice.

18. [Amazon Redshift](https://aws.amazon.com/redshift/)

- Store petabytes of structured or semistructured data in this data warehouse service. Analyze large datasets stored in the warehouse using complex SQL queries.

19. [AWS Glue Data Catalog](https://docs.aws.amazon.com/prescriptive-guidance/latest/serverless-etl-aws-glue/aws-glue-data-catalog.html)

- Provide metadata to various analytics services with this centralized repository.

20. [AWS Glue](https://aws.amazon.com/glue/)

- Process data by using the AWS Glue Data Catalog as a reference.

21. [Amazon EMR](https://aws.amazon.com/emr/)

- Process big data workloads by using popular frameworks like Apache Spark.

22. [Amazon Athena](https://aws.amazon.com/athena/)

- Analyze various data sources hosted anywhere with a single SQL query.

23. [Amazon QuickSight](https://aws.amazon.com/quicksight/)

- Visualize data by creating interactive dashboards with or without expertise.

24. [Amazon OpenSearch Service](https://aws.amazon.com/opensearch-service/)

- Visualize and monitor real-time data analytics with keyword or NLP searches.

# Module 9 - Security
25. [AWS Identity and Access Management (IAM)](https://aws.amazon.com/iam/)

- Securely manage identities and access to AWS services and resources.

26. [AWS IAM Identity Center](https://aws.amazon.com/iam/identity-center/)

- Connect your existing workforce identity source and centrally manage access to AWS with single sign-on.
27. [AWS Secrets Manager](https://aws.amazon.com/secrets-manager/)

- Centrally store and manage credentials, API keys, and other secrets.

28. [AWS Systems Manager](https://aws.amazon.com/systems-manager/)

- Manage nodes, or connection points, at scale on AWS and in multi-cloud and hybrid environments.

29. [AWS Shield](https://aws.amazon.com/shield/)

- Protect your network and applications from the most common, frequently occurring types of DDoS attacks.

30. [AWS WAF](https://aws.amazon.com/waf/)

- Protect your network and applications from blocked IP addresses defined by a web ACL.

31. [AWS Key Management Service (AWS KMS)](https://aws.amazon.com/kms/)

- Create and manage cryptographic keys to encrypt and decrypt your data.

32. [Amazon Macie](https://aws.amazon.com/macie/)

- Certify that sensitive data is discovered and protected in Amazon S3.

33. [AWS Certificate Manager (ACM)](https://aws.amazon.com/certificate-manager/)

- Create and manage SSL/TLS certificates that provide data encryption in transit.

34. [Amazon Inspector](https://aws.amazon.com/inspector/)

- Check applications for security vulnerabilities and deviations from security best practices.

35. [Amazon GuardDuty](https://aws.amazon.com/guardduty/)

- Continuously monitor the AWS environment with intelligent threat detection.

36. [Amazon Detective](https://aws.amazon.com/detective/)

- Analyze threats with interactive visualizations contained in a unified view.

37. [AWS Security Hub](https://aws.amazon.com/security-hub/)

- Aggregate security findings and organize them into actionable insights.
